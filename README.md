# IPPAN Foundation Website

The official website for the IPPAN Foundation, built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  **Build for production:**

    ```bash
    npm run build
    ```

## Deployment

To deploy on Vercel:

1.  Import this repository into Vercel.
2.  Deploy (no special configuration required, Vercel detects Next.js automatically).

## DevNet Explorer

The website links to the external DevNet Explorer: [https://ippan-explorer2.vercel.app/](https://ippan-explorer2.vercel.app/)

## Structure

-   `app/`: App Router pages and layouts
-   `components/`: Reusable UI components
-   `public/`: Static assets
